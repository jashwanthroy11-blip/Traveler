# Airy Adventure - Travel Planner & Boutique Explorer

Welcome to **Airy Adventure**! This is a premium, high-fidelity frontend travel application built with modern, minimalist design aesthetics, featuring elegant glassmorphism, responsive bento grids, and high-performance interactive components.

---

## 🌟 Key Features

1. **Boutique Explore Dashboard (`index.html`)**
   - Active experience tab filters (All Paths, Beach Escape, Mountain Path, City Explorer).
   - Dynamic destination grid with fluid card hovers.
   - Synchronized persistent favorites tracking.

2. **Custom Multi-Step Trip Planner (`create-trip.html`)**
   - 4-step wizard interface.
   - Dynamic calendar range selection highlighting start, end, and middle range spans.
   - Live night/day calculations based on selections.
   - AI travel designer simulator with beautiful loading states and custom itinerary rendering.

3. **User Profile & Milestones (`profile.html`)**
   - Real-time statistics counter tracks active liked cards.
   - Unit selection conversion (displays `12,450 Kilometers` or converts to `7,736 Miles` depending on preferences).
   - Interactive persistent grid.

4. **Preferences Dashboard (`settings.html`)**
   - General account preview.
   - Units configuration (Kilometers vs. Miles) persisted globally across views.
   - Beautiful simulated logout transitions.

## 🚀 Running Locally

The application runs directly in any modern web browser without needing a build step or complex dependency compilation:

1. Clone or open the workspace folder.
2. Double-click `index.html` to open it in your browser, or serve it using a lightweight local server extension (like Live Server).
3. Experience fluid navigation, persistent favorites, and dynamic unit configurations instantly!
